text-align — вирівнювання тексту. Будь-який текст вирівняний по лівому краю, але його можна вирівняти:

по центру:
text-align:center;

або по правому краю:
text-align:right

Есть ещё text-align: left, в нём особо нет логики, наверное поэтому не упоминается

font-size — висота букв тексту в пікселях: значення 15px – дрібний текст:15px, 50px – великий заголовок:50px.

font-family — шрифт: набір літер та символів, написаних у певному стилі. У сучасному веб-дизайні популярні шрифти без засічок: значення стилю sans-serif:

sans-serif

background —дозволяє встановити колір тла для сайту. Застосовується до тега ≤main≥, всередину якого міститься весь вміст сайту.Наприклад, поставимо колір фону світло-блакитний і шрифт без засічок:
